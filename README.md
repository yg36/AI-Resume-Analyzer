# AI Resume Analyzer

## Overview

Next.js resume analyzer that parses PDFs and uses OpenAI to score role fit, skills, gaps, and resume health.

Project type: JavaScript/React application.

## What This Repository Contains

- `README.md`
- `package.json`
- `src/app/api/analyze/route.js`
- `src/app/favicon.ico`
- `src/app/globals.css`
- `src/app/layout.js`
- `src/app/page.js`
- `src/components/ResultDashboard.jsx`
- `src/components/ResumeHealth.jsx`
- `src/components/ScoreCard.jsx`
- `src/components/SectionCard.jsx`
- `src/components/UploadResume.jsx`

## Tech Stack

Next.js, React

## How To Run / Use

```bash
npm install
npm run dev
```

For production validation, run:

```bash
npm run build
```
